# OpenClaw — публичная документация

Санитизированные версии рабочей документации по развёртыванию OpenClaw на VPS. Из файлов удалена приватная информация (домены, IP-адреса, токены, имена ботов, данные хостинга), сохранены архитектурные решения, требования безопасности и план развёртывания.

## Файлы

| Файл | Содержание |
|---|---|
| [openclaw_обзор_public.md](openclaw_обзор_public.md) | **Обзор платформы** — возможности, архитектура, безопасность, системные требования, установка, мониторинг, модель оплаты |
| [развёртывание_решение_public.md](развёртывание_решение_public.md) | **План развёртывания** — конкретные этапы (0–12), архитектурные решения, найденные проблемы и их решения, отложенные задачи |

## Как читать

- **Начните с обзора** (`openclaw_обзор_public.md`) — общее понимание платформы, её возможностей и рисков безопасности.
- **Затем план развёртывания** (`развёртывание_решение_public.md`) — пошаговая реализация с реальным опытом: какие проблемы возникли и как были решены.

## Что удалено

- Доменные имена, публичные и Tailscale IP-адреса
- Названия Telegram-ботов, токены, ID пользователей
- Конкретный хостинг-провайдер, тарифы, рублёвые цены
- Локальные пути с именами пользователей
- Операционные детали, специфичные для конкретной инсталляции
